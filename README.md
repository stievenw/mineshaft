## About

Mineshaft is an open-source game inspired by Minecraft. This project serves as an exploration into game development, procedural generation, and 3D graphics programming.



## Technology Stack

- **Language**: Java 21.0.8
- **Graphics Library**: LWJGL 3
- **Graphics API**: OpenGL
